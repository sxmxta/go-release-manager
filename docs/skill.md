name: energy-release
description: 该项目是基于energy + webview 实现的应用, 主要功能go项目的git仓库标签tag管理工具, 主要针对指定仓库添加，模块依赖继承，
父模块升级自动关联到子模块一同升级并自动创建新的版本标签,
这里的模块升级是指父模块项目在 git 仓库创建的新tag标签，并同步升级更新 go 项目, 继承链中的 go.mod 的模块需同步升级版本号。
注意：这里的操作都可自己维护可控制的所以不存在其它三方仓库。
